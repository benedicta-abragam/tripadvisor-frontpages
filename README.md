# Tripadvisor‑Frontpages

 A static front-end project recreating a travel experiences homepage using HTML and CSS.

## Overview
Tripadvisor‑Frontpages is a simple, responsive website built with *HTML and CSS* to showcase travel destinations, tours, and categories.
It focuses on layout, styling, and design.

## Features
- Homepage layout with header, navigation, and search section  
- Display of travel experiences with images and titles  
- Category sections like Outdoor, Food, Culture, Water  
- Responsive design, no JavaScript required

## Usage
1. Clone the repo:
   git clone https://github.com/benedicta-abragam/tripadvisor-frontpages.git    
2. Open index.html in your browser.

## Built With
- HTML5  
- CSS (Flexbox)
